# About d-money project
D-money is a transactional system where user can send money, payment and other transaction operation.

# Technology used:
- postman
- newman
- nodejs

# How to run?
- Clone this project
- PUT .env file
- Give following command:
- ``` npm i ```
- ``` npm test ```
